CentOs7 学习

网络管理命令

ping 命令不会自动停下，需要按ctrl +c 停止，或者用参数 -c 指定具体应答条数。

ping baidu.com -c 4

ifconfig

信息展示

<UP,BROADCAST,RUNNING,MULTICAST>RUNNING网卡设备已连接 MULTICAST支持组播

设置网卡的IP地址

ifconfig eno16777736:1 192.168.10.10 netmask 255.255.255.0 up ;给网卡eno16777736:1 的子接口1设置IP地址。

ifconfig可以禁用某个网络接口，或者更改MAC地址

复制文件至其他系统SCP

与远程主机交换文件可是使用FTP，或者搭建WEB服务外，还可以使用SCP, rsync

scp 将本地文件传送至远程主机或从远程主机拉取文件至本地

rsync是Linux系统下常用的数据镜像备份工具，用于不同主机之间同步文件，除了单个文件外，还可以镜像保存整个目录树和文件系统。

显示网络连接，路由表或接口状态netstat

监控系统网络配置和工作状态

netstat -a|head -4

netstat -at 显示所有TCP端口

netstat -au 显示所有UDP端口

netstat -i 显示所有接口的状态

探测至目的地址的路由信息traceroute

测试，登录或控制远程主机telnet 基于telnet协议

下载文件wget

-c 支持断点下载 ；强大的同时支持FTP或HTTP协议下载

Linux防火墙

默认firewalld ；iptables

禁用防火墙

systemctl stop firewalld ; systemctl disable firewalld

systemctl start iptables ;systemctl enable iptables

高级网络管理工具 iproute2

命令 ip

网络数据采集与分析工具 tcpdump

tcpdump -i any

tcpdump -i any tcp

动态主机配置协议DHCP

搭建DHCP DNS服务，安装相应的软件 服务端需要安装这些软件

路由表学习

命令 route

路由表属性列

Flags: 路由状态 U可用状态； G默认路由； C缓存的路由条目； ！拒绝路由；

Metric: 路由条目的代价，到此路由要经过的条数

一个实用的路由表要配置DHCP，数据包转发，NAT等。Linux要两个方案，一个是安装路由软件，如Zebra,二是自身配置。

静态路由和策略路由

文件共享服务

NFS Samba FTP

网络文件系统NFS 文件服务器Samba

NFS服务需要 rpcbind服务

命令mount

FTP服务基于TCP服务，使用两个端口，命令端口21，数据端口20 ；提示需要防火墙关闭或者设置白名单

FTP软件 vsftp proftpd purefrpd wuftpd

搭建LAMP服务 Linux Apache MySQL PHP

用途搭建WEB服务，或者电子商务需要用到的服务

Apache HTTP server 需要安装OpenSSL

Apache基于IP，端口，域名的虚拟主机配置 安全控制与认证

搭建LNMP服务 其中N代表Nginx

Nginx是一款轻量级的WEB服务软件，同时支持负载均衡和反向代理，并发能力强，国内用的居多。

Nginx依赖 PCRE包

Nginx与PHP集合方式有两种 一是通过spawn-fcgi； 一是 php-frm

集群

Linux集群体系

LVS负载均衡调度算法 ，LVS负载均衡的安装与设置

LVS框架，提供了IP虚拟服务器软件IPVS，包含了3种IP负载均衡技术

VS/NAT

VS/TUN

VS/DR

负载均衡通用体系

客户端 通过IP 访问负载均衡器 ，负载均衡器将客户端的请求 发送给合适的 真实服务器

高可用的MYSQL平台

可选方案HAProxy ,MySQL Proxy 和域名等。

高可用性集群技术

高可用软件Pacemaker keepalived

KVM虚拟机 和 oVirt虚拟化管理平台

virt-manager虚拟系统管理器

GlusterFS 开源分布式文件系统

与次同时还有MooseFS CEPH Lustre

OpenStack私有云

OpenNebula云平台
