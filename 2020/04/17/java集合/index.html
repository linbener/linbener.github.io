泛型

泛型需要用 菱形符号表示 <T>

泛型方法既可以存在泛型类当中，也可以存在普通类当中。

对泛型变量进行限定

<T extends Cat> 这个表明泛型T必须是Cat的子类，T的超类可以是Cat，也可以是间接继承Cat的类。

类型擦除

泛型的约束和局限

  1. Java不支持泛型类型的数组
  2. Varags警告 向方法传递一个可变类型的泛型参数就会产生这个警告
  3. 不能实例化泛型对象
  4. ….

通配符类型

<? extends Cat>

通配符的超类型限定

? super Cat

无限定的通配符 ？

在反射中使用的Class类是泛型的

String.class 实际上是一个Class类对象。

泛型类型的继承关系

超类Animal，有一个子类Cat ,而泛型Pets

在Pest 之间没有任何的继承关系Pest，虽然Cat的超类是Animal。

为了让泛型之间存在继承关系，用上通配符。

Pets <— Pest <—Pest ; Pest <—Pest

超类<—子类

Pets <— Pest <—Pest ; Pest <—Pest
