处理字节序列 InputStream OutputStream

丰富的流家族

操作文件

Files Path java7添加的新类

相比之前的流类，关心的是文件里面的内容，Files, Path操作的是文件整体。可以用来管理文件，复制文件，删除文件。

Path表示的是一个目录序列，后面也可以跟着一个文件名。路径中的第一个部件可以是根部件，那么创建的路径就是绝对路径，否则就是相对路径。

Unix 的/ ；windows的 C:/ 。

创建绝对路径

创建相对路径

静态的Paths.get()

路径中不必存在实际的文件，它只是一个抽象的路径，而Files类才是从路径创建文件的类。

对象 resolve方法 如果接受的路径是绝对路径，那就返回绝对路径，否则返回通过连接this和接受路径的路径。

resolveSibling()类似，它接受的相对路径

Files类可以使得普通文件的读写非常遍历

三个静态方法

copy move delete

前两个方法第三个参数为mode, 是一个枚举类StandardCopyOption

调用readAttributes方法

这个方法第二个参数为glob模式，可以用来设置过滤条件

缓存区数据结构

Java.nio能访问缓存结构，FileChannel 类 可以从文件获取一条通道，通道是用于磁盘文件的一种抽象，它可以让操作者 获取内存映射， 文件加锁机制 ， 文件间快速传递数据(可以用于进程间的数据交换) 等操作系统特性。

map方法获取内存映射，返回byteBuffer

父类Buffer 常用的 ByteBuffer CharBuffer 子类

多个同时执行的程序需要用到同一个文件

文件锁，控制文件或文件的某个范围的字节访问

file= FileChannel.open(path)

file.lock() file.tryLock()

网络

URL和URI

URL和URLConnection类封装了大量复制的实现细节，可以从远程站点获取信息，而不是向套字一样只是发送HTTP请求。

URL表示可以在互联网上定位到具体的资源，可以打开一个输入流。

URI类无法定位到任何的资源，也没有打开资源的方法，它的作用是 解析。它作用与Java类知道如何处理的模式，比如http, https, ftp ,file , jar….

使用URLConnection能比URL获得更多的信息

填写表单信息

可以用URLConnection方法的SetDoOutput(true)方法，建立一个输出的连接，然后获取getOutputStream方法获取输出流，用输出流参照表单提交HTTP的格式，往输出流填写数据，关闭输出流。

国际化

Locales

采用ISO国际标准，本地语言用两个小写字母，国家用两个大写字母表示。

比如中国language=zh,location=CN ,美国language=en,location=US

数字格式

不同国家的数字，货币，百分比的表示不同。

读取用户地区输入的数字

日期和时间

月份和星期应该用本地语言表示，年月日的顺序符合本地习惯，公历可能不是本地首选的日期表示方法，考虑本地的时区。这些是日期和时间格式化需要考虑的事情。

DataFormat类

格式，及日期时间表示的风格，用本地语言和本地习惯。
